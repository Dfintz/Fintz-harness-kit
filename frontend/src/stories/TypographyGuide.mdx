# Typography

The Star Citizen Fleet Manager design system uses a modern, readable typography system optimized for digital interfaces and long-form content.

## 📝 Font Families

### Primary Font
**System Font Stack**

We use a carefully curated system font stack for optimal performance and native feel:

```css
font-family: -apple-system, BlinkMacSystemFont, 'Segoe UI', 'Roboto', 
             'Helvetica Neue', Arial, sans-serif;
```

This provides:
- ✅ Zero load time (no web font downloads)
- ✅ Native OS appearance
- ✅ Excellent readability
- ✅ Wide character support

### Monospace Font
For code, data, and technical content:

```css
font-family: 'SF Mono', Monaco, 'Cascadia Code', 'Roboto Mono', 
             Consolas, 'Courier New', monospace;
```

## 📏 Type Scale

Our type scale follows a modular scale based on Adobe Spectrum design tokens:

### Headings

| Level | Size | Weight | Line Height | Usage |
|-------|------|--------|-------------|-------|
| H1 | 32px (2rem) | 700 | 1.2 | Page titles |
| H2 | 24px (1.5rem) | 600 | 1.3 | Section headers |
| H3 | 20px (1.25rem) | 600 | 1.4 | Subsection headers |
| H4 | 18px (1.125rem) | 600 | 1.4 | Card titles |
| H5 | 16px (1rem) | 600 | 1.5 | Small headings |
| H6 | 14px (0.875rem) | 600 | 1.5 | Labels |

### Body Text

| Size | Weight | Line Height | Usage |
|------|--------|-------------|-------|
| Large | 18px (1.125rem) | 500 | 1.6 | Emphasized body text |
| Regular | 16px (1rem) | 400 | 1.6 | Default body text |
| Small | 14px (0.875rem) | 400 | 1.5 | Secondary text |
| XSmall | 12px (0.75rem) | 400 | 1.4 | Captions, labels |

## 🎨 Text Colors

Colors are semantic and maintain WCAG AA contrast:

```css
/* Primary text */
color: #e0e8f0;

/* Secondary text */  
color: #8a9eb5;

/* Muted text */
color: #64748b;

/* Accent text */
color: #00d9ff;

/* Success text */
color: #00ff88;

/* Warning text */
color: #ffaa00;

/* Error text */
color: #ff4d4d;
```

## 💪 Font Weights

We use a limited set of weights for consistency:

- **400** - Regular (body text)
- **500** - Medium (emphasis)
- **600** - Semibold (headings, buttons)
- **700** - Bold (strong emphasis, H1)

## 📐 Spacing

### Paragraph Spacing
```css
margin-bottom: 1rem; /* 16px */
```

### Section Spacing
```css
margin-bottom: 1.5rem; /* 24px */
```

### Heading Margins
```css
margin-top: 2rem;     /* 32px */
margin-bottom: 0.5rem; /* 8px */
```

## 📱 Responsive Typography

Typography scales responsively:

```css
/* Mobile (< 768px) */
h1 { font-size: 24px; }
h2 { font-size: 20px; }
body { font-size: 14px; }

/* Tablet (768px - 1024px) */
h1 { font-size: 28px; }
h2 { font-size: 22px; }
body { font-size: 15px; }

/* Desktop (> 1024px) */
h1 { font-size: 32px; }
h2 { font-size: 24px; }
body { font-size: 16px; }
```

## 🔤 Special Formats

### Code
```css
font-family: monospace;
font-size: 0.875em;
padding: 0.2em 0.4em;
background: rgba(255, 255, 255, 0.1);
border-radius: 4px;
```

### Keyboard Keys
```css
font-family: monospace;
font-size: 0.875em;
padding: 0.2em 0.6em;
background: rgba(255, 255, 255, 0.1);
border: 1px solid rgba(255, 255, 255, 0.2);
border-radius: 4px;
```

### Links
```css
color: #00d9ff;
text-decoration: none;

&:hover {
  text-decoration: underline;
}
```

### Numbers (Tabular)
For tables and data displays:
```css
font-variant-numeric: tabular-nums;
```

## ♿ Accessibility

### Minimum Sizes
- Body text: 16px minimum
- Touch targets: 44x44px minimum
- Line height: 1.5 minimum for body text

### Contrast Requirements
All text meets WCAG 2.1 AA:
- Normal text (< 18pt): 4.5:1 minimum
- Large text (≥ 18pt): 3:1 minimum

### Readability
- Line length: 45-75 characters optimal
- Paragraph width: Max 65ch recommended

## 📝 Content Guidelines

### Writing Style
- **Clear and concise** - Use simple, direct language
- **Active voice** - "Save changes" not "Changes will be saved"
- **Sentence case** - For buttons, labels, and headers
- **Positive framing** - Focus on what users can do

### Punctuation
- **No periods** - In buttons, labels, single-line UI text
- **Use periods** - In paragraphs, help text, descriptions
- **Commas** - Oxford comma in lists

### Numbers
- **Spell out** - One through nine
- **Numerals** - 10 and above
- **Abbreviate** - Large numbers (1.2M, 5.3K)

### Date/Time
- **Relative** - "5 minutes ago", "Yesterday"
- **Absolute** - "Dec 3, 2025 at 2:30 PM"
- **24-hour** - For technical contexts
