# Star Citizen Fleet Manager Design System

Welcome to the Star Citizen Fleet Manager component library! This design system provides a comprehensive set of UI components built with **Adobe React Spectrum** and follows WCAG 2.1 AA accessibility standards.

## 🚀 Getting Started

Browse the component library in the sidebar to explore all available components. Each component includes:

- **Interactive examples** - Try different props and states
- **Documentation** - Detailed usage guidelines
- **Code examples** - Copy-paste ready code snippets
- **Accessibility notes** - WCAG compliance details

## 🎨 Design Principles

### Space-Themed Aesthetics

Our design system embraces the futuristic, space-themed aesthetics of Star Citizen with:

- **Dark color palette** with cyan and purple accents
- **Glass morphism effects** for depth and hierarchy
- **Glowing effects** for interactive elements
- **Responsive layouts** that work on all devices

### Accessibility First

Every component is built with accessibility in mind:

- ✅ WCAG 2.1 AA compliant color contrasts
- ✅ Keyboard navigation support
- ✅ Screen reader compatibility
- ✅ Focus management
- ✅ ARIA attributes

### Consistency

We maintain consistent patterns across:

- **Spacing** - Using Spectrum's spacing scale
- **Typography** - Clear hierarchy and readability
- **Colors** - Semantic color usage
- **Interactions** - Predictable behaviors

## 📦 Component Categories

### Core UI Components

Basic building blocks for your interface:

- **Button** - Primary, secondary, danger, ghost, outline variants
- **Input** - Text input with validation and states
- **Select** - Dropdown selection with search
- **Card** - Container for content grouping
- **Modal** - Dialog and modal overlays
- **DataTable** - Unified data table with sorting, pagination, selection & export (shared)

### Glass Components

Premium components with glass morphism effects:

- **GlassButton** - Translucent button with blur effects
- **GlassCard** - Glass-styled content container
- **Modal** - MUI Dialog-based modal (replaces GlassModal)
- **GlassPanel** - Large glass content panels

### Data Visualization

Components for displaying metrics and analytics:

- **StatCard** - Key metrics display
- **Sparkline** - Inline trend charts
- **TrendIndicator** - Trend direction and percentage
- **PeriodComparison** - Period-over-period comparison

### Utility Components

Helper components for common patterns:

- **EmptyState** - Empty state illustrations
- **LoadingSpinner** - Loading indicators
- **Toast** - Notification system
- **Typography** - Text styling utilities
- **Skeleton** - Loading placeholders

### Navigation

Components for navigation and wayfinding:

- **Breadcrumbs** - Hierarchical navigation
- **QuickActionCard** - Dashboard navigation cards

### Accessibility

Components for enhanced accessibility:

- **FocusTrap** - Focus management for modals

## 🎯 Usage Examples

### Basic Example

```tsx

  return (

      <CardHeader title="Fleet Overview" />

      </CardContent>
    </Card>

```

### With Toast Notifications

```tsx
import { Button } from '@/components/ui/Button';
import { useToast } from '@/components/ui/Toast';

function SaveButton() {
  const toast = useToast();

  const handleSave = async () => {
    try {
      await saveData();
      toast.success('Saved successfully!');
    } catch (err) {
      toast.error('Failed to save');
    }
  };

  return <Button onClick={handleSave}>Save</Button>;
}
```

### With DataTable

```tsx
import { DataTable } from '@/components/shared/DataTable';
import type { DataTableColumn } from '@/components/shared/DataTable';

interface Ship {
  id: string;
  name: string;
  manufacturer: string;
  crew: number;
}

function ShipList({ ships }: { ships: Ship[] }) {
  const columns: DataTableColumn<Ship>[] = [
    { id: 'name', label: 'Ship Name', sortable: true },
    { id: 'manufacturer', label: 'Manufacturer' },
    { id: 'crew', label: 'Crew Size', align: 'right', sortable: true },
  ];
  return (
    <DataTable
      columns={columns}
      data={ships}
      getRowId={ship => ship.id}
      defaultSort={{ column: 'name', direction: 'asc' }}
      pagination
    />
  );
}
```
