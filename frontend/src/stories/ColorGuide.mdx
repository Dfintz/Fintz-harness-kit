# Color Palette

The Star Citizen Fleet Manager design system uses a carefully crafted color palette that maintains accessibility while embracing the space-themed aesthetic.

## 🎨 Primary Colors

### Background Colors

- **Primary Background**: `#0a0e1a` - Main dark background
- **Secondary Background**: `rgba(15, 29, 53, 0.4)` - Semi-transparent panels
- **Tertiary Background**: `rgba(42, 63, 95, 0.4)` - Borders and dividers

### Text Colors

- **Primary Text**: `#e0e8f0` - Main text color (AA compliant)
- **Secondary Text**: `#8a9eb5` - Secondary/helper text (AA compliant)
- **Muted Text**: `#64748b` - Disabled/muted states

## 💫 Accent Colors

### Cyan (Primary Accent)
- **Value**: `#00d9ff`
- **Usage**: Primary actions, links, highlights
- **Background**: `rgba(0, 217, 255, 0.1)` - Hover/focus states
- **Border**: `rgba(0, 217, 255, 0.3)` - Borders

### Purple (Secondary Accent)
- **Value**: `#a855f7`
- **Usage**: Secondary actions, premium features
- **Background**: `rgba(168, 85, 247, 0.1)` - Hover/focus states
- **Border**: `rgba(168, 85, 247, 0.3)` - Borders

## ✅ Status Colors

All status colors meet WCAG 2.1 AA contrast requirements.

### Success
- **Value**: `#00ff88` / `#22c55e`
- **Usage**: Success messages, positive states
- **Background**: `rgba(0, 255, 136, 0.1)`

### Warning
- **Value**: `#ffaa00` / `#eab308`
- **Usage**: Warning messages, caution states
- **Background**: `rgba(255, 170, 0, 0.1)`

### Error
- **Value**: `#ff4d4d` / `#ef4444`
- **Usage**: Error messages, destructive actions
- **Background**: `rgba(255, 77, 77, 0.1)`

### Info
- **Value**: `#00d9ff`
- **Usage**: Informational messages
- **Background**: `rgba(0, 217, 255, 0.1)`

## 🎯 Interactive States

### Focus
- **Ring**: `0 0 0 3px rgba(0, 217, 255, 0.4)` - Focus indicator
- **Outline**: `2px solid #00d9ff` - Focus outline

### Hover
- Buttons: Opacity `0.9` or background lightening
- Links: Color shift to lighter shade
- Cards: Subtle glow effect

### Active/Pressed
- Buttons: Opacity `0.8` or background darkening
- Scale: `transform: scale(0.98)` for tactile feedback

### Disabled
- **Opacity**: `0.5`
- **Background**: Muted/grayed out
- **Cursor**: `not-allowed`

## 🌈 Glass Morphism

Glass components use these special effects:

### Glass Background
```css
background: rgba(15, 29, 53, 0.4);
backdrop-filter: blur(12px);
border: 1px solid rgba(255, 255, 255, 0.1);
```

### Glass Glow
```css
box-shadow: 
  0 8px 32px 0 rgba(0, 217, 255, 0.1),
  inset 0 1px 0 0 rgba(255, 255, 255, 0.1);
```

## 📊 Semantic Color Usage

### Data Visualization
- **Positive Growth**: `#00ff88`
- **Negative Growth**: `#ff4d4d`
- **Neutral**: `#8a9eb5`
- **Chart Lines**: Cyan, Purple, Orange, Green variants

### Ship Status
- **Active**: `#00ff88` (Green)
- **Maintenance**: `#ffaa00` (Orange)
- **Retired**: `#64748b` (Gray)
- **In Transit**: `#00d9ff` (Cyan)

### Priority Levels
- **Critical**: `#ff4d4d` (Red)
- **High**: `#ffaa00` (Orange)
- **Medium**: `#00d9ff` (Cyan)
- **Low**: `#8a9eb5` (Gray)

## ♿ Accessibility

All color combinations in this palette meet **WCAG 2.1 AA** standards:

- **Normal text**: Minimum 4.5:1 contrast ratio
- **Large text** (18pt+): Minimum 3:1 contrast ratio
- **UI components**: Minimum 3:1 contrast ratio

### Contrast Ratios

| Foreground | Background | Ratio | Status |
|------------|------------|-------|--------|
| #e0e8f0 | #0a0e1a | 11.2:1 | AAA |
| #8a9eb5 | #0a0e1a | 5.8:1 | AA |
| #00d9ff | #0a0e1a | 8.2:1 | AAA |
| #00ff88 | #0a0e1a | 11.7:1 | AAA |
| #ff4d4d | #0a0e1a | 5.1:1 | AA |

## 🎨 CSS Variables

Colors are available as CSS custom properties:

```css
/* Backgrounds */
--bg-primary: #0a0e1a;
--bg-secondary: rgba(15, 29, 53, 0.4);
--bg-tertiary: rgba(42, 63, 95, 0.4);

/* Text */
--text-primary: #e0e8f0;
--text-secondary: #8a9eb5;
--text-muted: #64748b;

/* Accents */
--accent-cyan: #00d9ff;
--accent-purple: #a855f7;

/* Status */
--success: #00ff88;
--warning: #ffaa00;
--error: #ff4d4d;
--info: #00d9ff;

/* Borders */
--border-subtle: rgba(255, 255, 255, 0.1);
--border-default: rgba(42, 63, 95, 0.4);
--border-strong: rgba(255, 255, 255, 0.2);
```

## 🌙 Dark Mode Only

This design system is **dark mode only** to match the Star Citizen aesthetic and provide optimal viewing experience for the space-themed interface. The dark background reduces eye strain during extended use and enhances the luminous accent colors.

## 📝 Usage Guidelines

### Do's ✅
- Use high contrast for important text
- Apply accent colors consistently across the app
- Use status colors for their semantic meaning
- Maintain glass effects for premium components

### Don'ts ❌
- Don't use light backgrounds
- Don't use accent colors for status indicators
- Don't mix status color meanings
- Don't reduce contrast below AA standards
